# EDA-on-employee-data-set
Exploratory Data Analysis or EDA involves the process of analysing general pattenrs in data, summarizing main characteristics and gaining insights from it
The steps involved in this project are:
1. Importing the required python libraries
2. Reading the dataset
3. Data Analysis
4. Check for duplication
5. Missing value calculation
6. Data cleaning
7. Exploratory Data Analysis
8. Statistical summary
9. Data visualization
10. Conclusion
